The Deep Learning classifier challenge


Some wind turbines drivetrain faults can be identified analysing signals recorded from accelerometers in each of the 3D axis: X, Y and Z.

You are challenged to develop a model that identifies each of the 7 fault classes induced in a turbine drivetrain.

The model, defined in scikit-learn or Keras format can be submitted here to be evaluated and ranked against other models.

To train the model a dataset is available here.

The train dataset has a set of samples, each sample with information from 20 seconds of turbine operation recorded by a 3D accelerometer setup on the drivetrain.

The dataset is stored in pickle format, with 2 variables: X and y. The X variable is a Numpy array with dimension Nx6000, where N is the number of samples, each with 6000 points. The first 2000 points are from accelerometer channel 0, the x-axis accelerometer, the next 2000 points from channel 1, the y-axis and the last from channel 2, the z-axis:

The y variable is also a Numpy array with dimension Nx7. For each sample there is an element in y, where the 7 fault classes are one hot encoded.

This layout, as is stored in the file, is suitable for a 1D model. For 2 D models different layouts can be formed after loading the data. For instance, data from accelerometer channels x,y and z can be stacked:

To load a dataset from a file, say 'dataset.pickle', the following code snippet can be used:

	import pickle, numpy
	from numpy.typing import NDArray

	fname = 'dataset.pickle'

	with open(fname, 'rb') as f:
	    X : NDArray = pickle.load(f)
	    y : NDArray = pickle.load(f)

	print("Number of samples:",     X.shape[0]) # or y.shape[0]
	print("Points in each sample:", X.shape[1])
	print("Fault classes:",         y.shape[1])
